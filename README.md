# Earthquake Intuition and Event Comparison Log

This repository is used to record my personal earthquake-related intuitions, warnings, or sensations at different times, and to compare them with major earthquakes that later occurred. The main goal is to build a long-term record and observe whether there is any recurring time interval or pattern between the date of an intuition and the date of a later earthquake event.

> Note: This is a personal experience log and retrospective comparison record. It is not a scientific earthquake forecast and should not be used as public safety guidance. Earthquake magnitudes, casualties, and impacts should be checked against reliable sources and updated over time.

## Recording Method

Each entry should keep as much of the following information as possible:

- Date of intuition / warning
- Subjective description at the time
- Later earthquake event that may correspond to it
- Earthquake date, location, magnitude, and impact
- Time interval between the intuition and the event
- Confidence level, uncertainties, and items that still need verification

# 地震感知与事件对照记录

这个仓库用于记录我个人在不同时间出现的地震相关感知、预警或直觉，以及后续实际发生的较大地震事件。主要目标是长期积累记录，观察“感知出现时间”和“实际地震发生时间”之间是否存在稳定的时间间隔或其他关联。

> 说明：这里记录的是个人经历和事后对照，不构成科学地震预报，也不作为公共安全建议。涉及地震规模、伤亡和影响的数据后续需要尽量补充可靠来源并持续修正。

## 记录方式

每条记录尽量保留以下信息：

- 感知 / 预警日期
- 当时的主观描述
- 后续可能对应的地震事件
- 地震发生日期、地点、震级和影响
- 感知与事件之间的时间间隔
- 可信度、疑点和待核验内容
